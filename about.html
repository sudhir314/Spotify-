About This Project

Project Overview

This is a functional Spotify clone built from the ground up to demonstrate full-stack web development skills. It features a custom backend API and a dynamic frontend that allows for a seamless music playing experience.

Core Features

  • Dynamic music library loaded from a database.
  • Permanent file storage for audio and images using Cloudinary.
  • A private admin dashboard for uploading and deleting songs.
  • Full audio controls: play/pause, next/previous, progress bar, and volume.
  • Live timers displaying current and total song duration.

Technologies Used

Frontend: HTML5, CSS3, Vanilla JavaScript (ES6+)
Backend: Node.js, Express.js
Database: MongoDB with Mongoose
Deployment: Render (Backend), Netlify (Frontend), MongoDB Atlas (Database)
File Storage: Cloudinary
